# IT User Lifecycle Automation Tool

## 📌 Overview
This project simulates user lifecycle management in an IT environment, including onboarding and offboarding processes. It is inspired by real-world IT infrastructure operations such as Active Directory and SaaS user provisioning.

---

## ⚙️ Features
- Reads user data from a CSV file
- Simulates account creation (Joiners)
- Simulates account deactivation (Leavers)
- Logs all operations with timestamps
- Demonstrates automation used in IT operations

---

## 🛠️ Technologies Used
- Python
- CSV file handling
- Basic logging
